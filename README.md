# 自动学习

## 使用方法

1. windows安装微信并登录
1. 关注**青春上海**公众号
1. 截取微信左上角头像替换image/head.png
1. `git clone https://github.com/jqiuyin/fk_learning.git`
1. `cd fk_learning`
1. `pip install -r requirements.txt`
1. 运行`python main.py`

## 运行要求

1. windows
1. python3.8
1. PyAutoGUI==0.9.53

## 注意事项

1. 仅在2560x1080分辨率的屏幕上测试过
1. 仅支持windows新系统
